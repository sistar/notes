Sublime Repl Clojure on Linux 101

essential keyboard shortcuts:
ALT 1..N 	window
STRG 1..N 	focus group 
STG SHIFT p	Command palette => type cloj to get to clojure repl
Eval in Repl (will only work from clj file)
	STRG SHIFT , + l lines
	STRG SHIFT, + f file
	STRG SHIFT, + s selection
	STRG SHIFT, + b block

Fuzzy Search stuff
STRG i incremental search
STRG d multiple cursors
STRG p goto arbitrary file, and continue to symbol using @
STRG r goto symbol
